## 🔧 技术亮点

### 1. Adapter 模式

通过 PlanningInterface 实现模型无关：

```python
# 统一的接口
class BasePlanningAdapter:
    def extract_scene_token() -> torch.Tensor
    def extract_reference_plan() -> torch.Tensor
    def extract_plan_confidence() -> torch.Tensor
    def extract_safety_features() -> Dict
```

### 2. 快速集成策略

同类框架复用，大幅降低工作量：

- **VAD → VADv2**: 2小时（直接复用）
- **SparseDrive → SparseDriveV2**: 2小时（直接复用）
- **DiffusionDrive → DiffusionDriveV2**: 30分钟（直接复用）

### 3. 统一的训练流程

```bash
# 所有模型使用相同的训练命令
python scripts/train_evaluator.py --data_dir data/vad_dumps
python scripts/train_evaluator.py --data_dir data/sparsedrive_dumps
python scripts/train_evaluator.py --data_dir data/uniad_dumps
```
